## Ablauf: Schritt-fuer-Schritt

### Phase 1: Vorbereitung (KW 8-9, Februar 2026)

- [ ] **Agentur fuer Arbeit anrufen**: 0800 4 555 500
  - Beratungsgespraech zum Gruendungszuschuss vereinbaren
  - Restanspruch ALG I in Tagen berechnen lassen
  - Formular "Anforderung der Stellungnahme" erhalten
- [ ] **Businessplan fertigstellen**: Vorlage (01-businessplan.md) mit persoenlichen Daten befuellen
- [ ] **Finanzplanung pruefen**: Alle Zahlen auf eigene Situation anpassen
- [ ] **Lebenslauf erstellen**: Vorlage (07-lebenslauf-vorlage.md) befuellen
- [ ] **Zeugnisse/Zertifikate kopieren**: Alle relevanten Qualifikationsnachweise

### Phase 2: IHK-Einreichung (KW 10-11, Maerz 2026)

- [ ] **Alle Unterlagen zusammenstellen** (ungeheftet)
- [ ] **Per Post oder persoenlich** an IHK Berlin einreichen
- [ ] **Umschlag beschriften**: GRUENDUNGSZUSCHUSS
- [ ] **Gebuehr**: 50,00 EUR + MwSt. bei Abholung bezahlen

### Phase 3: Stellungnahme (KW 12-14, Maerz 2026)

- [ ] **IHK prueft** (ca. 2-3 Wochen Bearbeitungszeit)
- [ ] **Stellungnahme abholen** + Vereinbarung unterschreiben
- [ ] **Gebuehr bezahlen**: 50,00 EUR + MwSt.

### Phase 4: Antragstellung (KW 14-15, Ende Maerz / Anfang April 2026)

- [ ] **Alle Unterlagen + IHK-Stellungnahme** bei Agentur fuer Arbeit einreichen
- [ ] **Gewerbeanmeldung** am vereinbarten Gruendungsdatum (01.04.2026)
- [ ] **Agentur fuer Arbeit entscheidet** (ca. 2-4 Wochen)

## Wichtige Hinweise

1. **NICHT vor Antragstellung das Gewerbe anmelden!** Die Gewerbeanmeldung erfolgt erst am vereinbarten Gruendungsdatum.
2. **150-Tage-Regel beachten**: Bei Gruendung muessen noch mindestens 150 Tage Restanspruch auf ALG I bestehen.
3. **Budget-Pools der Agenturen**: Fruehe Antragstellung (Q1) hat erfahrungsgemaess hoehere Bewilligungsquoten.
4. **Hauptberufliche Selbstaendigkeit**: Mindestens 15 Stunden/Woche.
5. **IHK prueft nur gewerbliche Gruendungen** (keine Freiberufler, keine Handwerker -- dafuer Handwerkskammer).
