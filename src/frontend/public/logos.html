4 directions. Pick the one that feels right for the brand.

Concept 01
The Accent Mark
A C-shield shape with a cyan accent bar cutting through. Compliance = protection. The open arc suggests both a shield and a data flow. Minimal, works at any size.
Light background variant
Concept 02
The Hash Chain
Three interlocking chain links forming a C-shape. References the SHA-256 hash chain — Caelith's core differentiator. Every link is a verified decision. Ascending opacity dots suggest data flow.
Concept 03
The Threshold Gate
Two pillars with a glowing horizontal threshold line. Visual metaphor for eligibility checks — above the line = approved, below = blocked. The arrow suggests data being validated in real-time.
Concept 04
The Data Monogram
A geometric C built from solid bars, with a data grid pattern inside. The grid represents compliance data flowing through the system — some green (passed), some red (flagged). The most "techy" option. Looks like a terminal meets a compliance dashboard.
